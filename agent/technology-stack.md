# Technology Stack

## Runtime
* **Browser JavaScript** — the app runs locally in a browser (no server required).

## Language
* **TypeScript 5.x** — strict mode enabled. Compiles to ES2022.

## Build Tooling
* **Vite 6.x** — dev server and production bundler.
* **Node.js 20+** — build-time only (not a runtime dependency).

## Testing
* **Vitest 3.x** — unit and integration tests. Compatible with Vite config.
* **Playwright 1.x** — E2E browser tests.

## Linting & Formatting
* **ESLint 9.x** — flat config, TypeScript plugin.

## UI
* **Vanilla DOM + HTML Canvas** — no framework. Canvas is used for the glyph pixel editor.

## Package Manager
* **npm** — lockfile committed.
